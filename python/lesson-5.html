📚 Урок 5: Цикли (Loops)

Що таке цикл?

Цикл — це конструкція, яка повторює блок коду кілька разів. Це економить час і робить код більш ефективним.

Цикл for

для і у range(5):
    print(i) # виведе 0, 1, 2, 3, 4

для фрукт у фрукти:
    print(фрукт)

Цикл while

і = 0
поки i < 5:
    print(i)
    i = i + 1
💡 Важливо: Переконайтесь, що цикл while коли-небудь завершиться, інакше вона зависне!
