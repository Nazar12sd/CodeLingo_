📚 Урок 5: Цикли (for, while)

Що таке цикл?

Цикл — це конструкція мови програмування, яка дозволяє повторювати виконання певного блоку коду кілька разів. В Python є два основних типи циклів: for та while.

Цикл for

Цикл for використовується для ітерування (проходження) по послідовності елементів, таких як список, рядок або діапазон чисел.

Синтаксис циклу for:

for змінна in послідовність:
Команда

Приклад використання циклу for:

for i in range(3):
print(i)

Вивід:

0
1
2

У цьому прикладі цикл for ітерує по діапазону чисел від 0 до 2 (3 не включно) і виводить кожне число на екран. Змінна i по черзі приймає значення з цієї послідовності.

Функція range() створює послідовність чисел, по якій зручно ітеруватися в циклі for.

range(5) # 0 1 2 3 4
range(2, 6) # 2 3 4 5
range(1, 10, 2) # 1 3 5 7 9

Цикл while

Цикл while використовується для повторення блоку коду, поки певна умова є істинною.

Синтаксис циклу while:

while умова:
Команда

Приклад використання циклу while:

i = 0
while i < 3:
print(i)
i = i + 1

Вивід:

0
1
2

У цьому прикладі цикл while повторюється, поки змінна i менша за 3. На кожній ітерації виводиться поточне значення i, а потім i збільшується на 1.

Коли використовувати for, а коли while?

Використовуйте цикл for, коли вам потрібно ітерувати по послідовності елементів або виконати блок коду певну кількість разів. Використовуйте цикл while, коли ви хочете повторювати блок коду, поки певна умова є істинною, і кількість повторень невідома заздалегідь.

💡 Запам'ятай: Цикл for використовується, коли ви знаєте кількість ітерацій заздалегідь, а цикл while - коли кількість ітерацій невідома.
