📚 Урок 2: Змінні і типи даних

Що таке змінна?

Змінна — це контейнер для зберігання даних. Уявіть, що це коробка, у якій ми можемо зберігати інформацію.

ім_я = "Іван"
вік = 13
зріст = 1.65

Ось ми створили три змінні: ім_я, вік та зріст.

Типи даних

У Python існує кілька основних типів даних:

  • String (рядок): текст, наприклад "Привіт"
  • Integer (ціле число): числа без десяткової частини, наприклад 42
  • Float (дійсне число): числа з десяткову частину, наприклад 3.14
  • Boolean (логічне значення): True або False

Приклади змінних

назва_школи = "Школа №1" # String
клас = 7 # Integer
оцінка = 4.5 # Float
здатен = True # Boolean
💡 Порада: Назви змінних повинні бути зрозумілими. Замість x, a, b, пишіть вік, ім_я, оцінка.

Операції зі змінними

a = 10
b = 5
сума = a + b # 15
різниця = a - b # 5
добуток = a * b # 50

Висновок

Тепер ви знаєте про змінні та типи даних. Це основні будівельні блоки програмування!
