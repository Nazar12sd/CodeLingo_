📚 Урок 1: Знайомство з Python

Що таке Python?

Python — це мова програмування, яка дозволяє:

  • Створювати ігри
  • Робити сайти
  • Писати ботів
  • Працювати з ШІ

Python читається майже як звичайна англійська, тому він ідеальний для початку.

Що таке програма?

Програма — це алгоритм, записаний спеціальною мовою та призначений для виконання комп'ютером. Python читає ці команди зверху вниз.

Приклад:

print("Hello World!")

Вивід:

Hello World!

Цей код просто виведе на екран текст Hello World!.

Перша команда — print()

💡 Запам'ятай: У Python функція print() виводить текст або число на екран.

Важливо:

  • Текст завжди в " ". Без лапок буде помилка
  • Числа виводяться без лапок
  • Числа може бути отримане

Неправильно:

print(Hello World!)

Правильно:

print("Hello World!")

Якщо писати:

print("Hello")
print("Word!")

Виведе:

Hello
Word!

Hello World! вивелося на різних рядках, тому що кожна команда print() в Python за замовчуванням виводить текст з перенесенням на новий рядок.

💡 Запам'ятай:

Що пишемо Що отримаємо
print("Привіт") Виведе текст
print(5) Виведе число 5
print("5") Виведе текст 5
print(Hello) ❌ Помилка, якщо це не ім'я змінної

Приклад:

print("Привіт, світ!")
print(5)
print("5")

Вивід:

Привіт, світ!
5
5

Також можна виводити текст і число в одній команді, розділяючи їх комами.

Приклад:

print("У мене", 5, "яблук")

Вивід:

У мене 5 яблук
