📚 Урок 1: Знайомство з Python

Рівень складності: Початківець

Що таке Python?

Python — це мова програмування, яка дозволяє:

  • Створювати ігри
  • Робити сайти
  • Писати ботів
  • Працювати з ШІ

Python читається майже як звичайна англійська, тому він ідеальний для початку.

Що таке програма?

Програма — це алгоритм, записаний спеціальною мовою та призначений для виконання комп'ютером. Python читає ці команди зверху вниз. Приклад:

print("Hello World!")

Цей код просто виведе на екран текст "Hello World!".

Перша команда — print()

💡 Запам'ятай: У Python функція print() виводить текст на екран.

Важливо:

  • Текст завжди в " "
  • Без лапок буде помилка

Неправильно:

print(Hello World!)

Правильно:

print("Hello World!")

Якщо писати:

print("Hello")
print("Word!")

Виведе:

Hello
Word!

"Hello World!" вивелося на різних рядках, тому що кожна команда print() в Python за замовчуванням виводить текст з перенесенням на новий рядок.
