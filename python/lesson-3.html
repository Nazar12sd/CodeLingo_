📚 Урок 3: Умови (if-else-elif)

Що таке Умова?

Умова — це перевірка, яка може бути:

  • Істина (True)
  • Хибна (False)

Умовні оператори дозволяють виконувати різні дії залежно від певних умов. В Python для цього використовуються ключові слова if, elif та else.

Команда if

В Python умова записується так:

if Умова:
Команда

Якщо умова правильна — команда виконається.

Якщо неправильна — команда не виконається.

Приклад:

age = 20
if age > 18:
print("Ви повнолітні")

Вивід:

Ви повнолітні

Як бачимо вивело "Ви повнолітні" бо змінна age дорівнює 20, що більше за 18 і умова виконалась. Якби змінна age була меншою або дорівнювала 18, умова була б хибною і команда не виконалась.

Команда else

Команда else виконується, якщо умова if є хибною.

В Python else записується так:

if Умова:
Команда
else:
Команда

Ось приклад як використовувати else:

age = 16
if age > 18:
print("Ви повнолітні")
else:
print("Ви не повнолітні")

Вивід:

Ви не повнолітні

Команда elif

Команда elif (else if) дозволяє перевіряти додаткові умови, якщо попередня умова є хибною.

В Python elif записується так:

if Умова:
Команда
elif Умова:
Команда
else:
Команда

Ось приклад як використовувати elif:

age = 18
if age > 18:
print("Ви повнолітні")
elif age == 18:
print("Ви саме повнолітні")
else:
print("Ви не повнолітні")

Вивід:

Ви саме повнолітні
💡 Запам'ятай: Оператори порівняння:
  • > — більше
  • < — менше
  • >= — більше або дорівнює
  • <= — менше або дорівнює
  • == — дорівнює
  • != — не дорівнює

Логічний оператори

Логічний оператор and повертає True, якщо обидва вирази є істинними.

a = 5
b = 10
if a > 3 and b < 15:
print("Обидва умови виконані")

Логічний оператор or повертає True, якщо хоча б один з виразів є істинним.

a = 5
b = 16
if a > 3 or b < 15:
print("Хоча б одна умова виконана")

Логічний оператор not повертає True, якщо вираз є хибним, і навпаки.

a = 2
if not a > 3 :
print("Умова не виконана")
