📚 Урок 8: Файли та вхід/вихід

Читання та запис файлів

Python дозволяє легко читати та записувати файли. Це корисно для збереження даних.

Запис у файл

файл = open("текст.txt", "w") # відкрити для запису
файл.write("Привіт, світ!")
файл.close() # закрити файл

Читання з файлу

файл = open("текст.txt", "r") # відкрити для читання
зміст = файл.read()
print(зміст)
файл.close()

Режими файлів

  • "r" - читання
  • "w" - запис (перепишує файл)
  • "a" - додавання (дописує в кінець файлу)
  • "x" - створення (помилка, якщо файл існує)
💡 Важливо: Завжди закривайте файли після роботи з ними! Або використовуйте конструкцію with.
