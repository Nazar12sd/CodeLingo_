📚 Урок 6: Списки (list) у Python

Що таке список?

Список (list) - це тип даних, який дозволяє зберігати декілька значень в одній змінній. Наприклад, якщо потрібно зберегти 5 чисел — не потрібно створювати 5 змінних.

Замість цього використовується список:

my_list = [1, 2, 3, 4, 5]

В цьому прикладі my_list - це список, який містить п'ять елементів: 1, 2, 3, 4 та 5.

Створення списку

Список створюється за допомогою квадратних дужок []. Елементи списку розділяються комами.

numbers = [1, 2, 3, 4, 5]
fruits = ["Яблоко", "Банан", "Апельсин", "Груша", "Киві"]

У списку можуть бути:

  • Числа (цілі та з плаваючою крапкою)
  • Рядки (текст в лапках)
  • Булеві значення (True або False)
  • Навіть змішані типи

mixed = [10, "привіт", 3.14]

Індекси у списку

Кожен елемент у списку має свій індекс, який використовується для доступу до нього. Нумерація починається з 0.

my_list = [1, 2, 3, 4, 5]
first_element = my_list[0] # 1
last_element = my_list[4] # 5

Також можна використовувати від'ємні індекси для доступу до елементів з кінця списку:

my_list = [1, 2, 3, 4, 5]
last_element = my_list[-1] # 5
second_last = my_list[-2] # 4

Зміна елемента списку

Елемент можна змінити за індексом. Наприклад, щоб змінити перший елемент списку:

my_list = [1, 2, 3, 4, 5]
my_list[0] = 10 # Змінюємо перший елемент

Результат:

my_list = [10, 2, 3, 4, 5]

Довжина списку — функція len()

Щоб дізнатися кількість елементів у списку, використовується функція len().

my_list = [1, 2, 3, 4, 5]
length = len(my_list) # 5

Вивід:

5

Перебір списку циклом for

Щоб виконати дію для кожного елемента списку, використовується цикл for.

my_list = [1, 2, 3, 4, 5]
for element in my_list:
  print(element)

Вивід:

1
2
3
4
5

Додавання елемента — append()

Щоб додати новий елемент в кінець списку, використовується метод append().

my_list = [1, 2, 3, 4, 5]
my_list.append(6) # Додаємо новий елемент в кінець списку

Результат:

my_list = [1, 2, 3, 4, 5, 6]

Видалення елемента — remove()

Щоб видалити елемент зі списку за значенням, використовується метод remove().

my_list = [1, 2, 3, 4, 5]
my_list.remove(3) # Видаляємо елемент зі значенням 3

Результат:

my_list = [1, 2, 4, 5]

Видалення останнього елемента — pop()

Щоб видалити останній елемент зі списку, використовується метод pop().

my_list = [1, 2, 3, 4, 5]
my_list.pop() # Видаляємо останній елемент

Результат:

my_list = [1, 2, 3, 4]

Видалення елемента за індексом — del

Щоб видалити елемент зі списку за індексом, використовується оператор del.

my_list = [1, 2, 3, 4, 5]
del my_list[2] # Видаляємо елемент з індексом 2

Результат:

my_list = [1, 2, 4, 5]
💡 Запам'ятай: Команди для роботи зі списками:
  • len() — повертає кількість елементів у списку
  • append() — додає елемент до кінця списку
  • remove() — видаляє елемент за значенням
  • pop() — видаляє елемент за індексом
  • del — видаляє елемент за індексом
